# 订货单生成器 — 使用说明

> 这个程序帮你把照片和货号信息变成一张专业的订货单表格。

## 第一步：准备好照片和信息

1. 用手机拍好每个产品的照片
2. 把照片传到电脑上，放到 **同一个文件夹** 里（比如放在桌面新建一个"照片"文件夹）
3. 准备好每个产品的 **货号** 和 **单价**

## 第二步：打开程序

双击 `订货单生成器.exe`，等待几秒钟，会出现一个窗口。

## 第三步：选择模板（可以跳过）

- 点击「选择模板」按钮
- 选一个以前做好的订货单文件（.xlsx）
- 程序会按照这个格式来生成新的订货单
- **如果跳过这步**，程序会用默认格式

## 第四步：选择照片文件夹

- 点击「选择照片」按钮
- 找到刚才放照片的文件夹，选中它
- 照片会显示在窗口中

## 第五步：粘贴产品信息

在右侧的大文本框中，把产品的信息**复制粘贴**进去。支持三种格式：

**格式1：从Excel复制**
```
品名    编码    件数    单价
智能手表  SW-1001  50    899
蓝牙耳机  BE-2022  120   199
```

**格式2：每段一个产品（空行隔开）**
```
货号：A-001
品名：LED灯泡
编码：LB-2401
件数：5件
每件数量：240个
单价：7元
重量：8.5KG

货号：B-002
品名：节能灯管
单价：15元
一件240个
```

**格式3：一行一个产品**
```
货号：A-001 单价：10元 一件100个
货号：B-002 单价：15元 一件50个
```

程序会自动识别填了哪些字段，没填的留空。

## 第六步：生成订单

- 检查信息都填好了
- 点击底部「生成订单」按钮
- 文件会自动保存到 **电脑桌面**
- 文件名类似：`订货单 2026年5月28日.xlsx`

## 常见问题

**Q: 提示"请选择照片文件夹"怎么办？**
A: 还没有选照片，点「选择照片」按钮选文件夹。

**Q: 提示"请粘贴订货信息"怎么办？**
A: 还没有粘贴产品信息，把微信上的信息复制后贴到文本框里。

**Q: 生成的文件在哪里？**
A: 在电脑桌面上，文件名以"订货单"开头。

**Q: 照片顺序乱了怎么办？**
A: 照片按文件名的顺序排列，可以在文件夹里把照片重命名成 1、2、3...来排序。

**Q: 提示"数量不匹配"怎么办？**
A: 照片数量和识别出的产品数量不一致，检查一下是不是少贴或多贴了某个产品。
