# 订货单生成器 — 项目方案

## 为什么做这个项目

义乌商贸城摊位主需要频繁生成带照片的订货单。目标是把这件事变成**一键操作**，让完全不懂电脑的人也能用。

## 核心功能

1. **上传模板**：传一个已有的订货单 .xlsx，程序自动识别格式
2. **选照片**：选择产品照片文件夹
3. **粘贴文本**：将微信/Excel上的产品信息粘贴到文本框，程序自动解析
4. **一键生成**：输出格式与模板一致的订货单

## 技术选型

| 需求 | 方案 | 理由 |
|------|------|------|
| 桌面程序 | Python + Tkinter | 内置库，无需额外安装 |
| 打包分发 | PyInstaller → 单文件 .exe | 双击即用，无需 Python 环境 |
| Excel 读写 | openpyxl | 支持公式、样式、图片 |
| 图片处理 | Pillow | 缩略图生成、缩放 |
| 文本解析 | 正则表达式 | 支持多种输入格式（表格、多行key-value、单行） |

## 用户操作流程

1. 双击打开程序
2. 点「选择模板」— 选一个已有的订货单文件（也可跳过，用默认格式）
3. 点「选择照片」— 选装好照片的文件夹
4. 在文本框中粘贴产品信息（支持Excel复制、key-value等多种格式）
5. 点「生成订单」— 文件保存到桌面
