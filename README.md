# 订货单生成器 — 义乌商贸城零门槛订单工具

> 粘贴文本 → 自动解析 → 一键生成带产品图的 xlsx 订单。给不懂电脑的家人用。

## 它能干什么

把批发市场的订货流程从「手写单据」变成「粘贴即生成」。零门槛，零学习成本。

| 步骤 | 你做什么 | 软件自动做什么 |
|------|---------|-------------|
| **选模板** | 点击选择（可选） | 加载你的 xlsx 订货单模板 |
| **选照片** | 选择照片文件夹 | 自动匹配产品图片 |
| **填信息** | 粘贴货号、单价、件数 | 自动解析文本，提取货号/单价/件数 |
| **生成订单** | 点击生成 | 生成带产品图的 xlsx 订货单 |

## 为什么做这个项目

家里在义乌国际商贸城经营科技消费品摊位（充电器、数据线、耳机等）。客户来看货时，需要快速生成带产品照片的订货单。

之前用 Excel 手动填，效率低、容易出错，家人也不太会操作复杂的表格软件。

最初想做微信小程序，但审核周期长。先做了桌面版，家人双击就能用，不需要联网，不需要安装任何东西。

## 功能特性

- **零门槛** — 双击运行，不需要安装 Python 或任何依赖
- **粘贴解析** — 直接粘贴文本，自动识别货号、单价、件数
- **产品图片** — 自动从文件夹匹配产品图片，插入订单
- **模板驱动** — 支持任意 xlsx 格式的订货单模板
- **一键生成** — 填完信息，点击按钮立即生成订单
- **离线使用** — 无需联网，数据本地处理

## 适用人群

- 义乌商贸城等批发市场的档口经营者
- 需要快速生成带照片的订货单
- 完全不懂电脑的人也能用

## 快速开始

### 方式一：下载 exe（推荐）

1. 从 Releases 下载 `订货单生成器.exe`
2. 双击运行
3. 选择模板（可选，首次使用会提示选择）
4. 选择照片文件夹
5. 粘贴货号、单价信息
6. 点击「生成订单」

### 方式二：源码运行

```bash
# 克隆仓库
git clone https://github.com/YaaLO-o/hub.git
cd hub

# 安装依赖
pip install -r requirements.txt

# 运行
python src/main.py
```

## 使用技巧

### 粘贴格式

软件支持多种粘贴格式，常见的有：

```
货号 单价 件数
A001 25.00 100
A002 18.50 200
```

或者直接从 Excel 复制的表格数据。

### 照片匹配

- 照片文件名需要包含货号（如 `A001.jpg`、`数据线_A002.png`）
- 支持 jpg、png、jpeg 格式
- 软件会自动匹配并插入订单

### 模板定制

- 首次使用时选择你的 xlsx 模板
- 模板保存后，后续使用自动加载
- 支持任意 xlsx 格式

## 常见问题

**Q: 打不开 exe 怎么办？**
A: 可能是 Windows 安全警告，右键选择「仍然运行」即可。

**Q: 照片没有插入？**
A: 检查照片文件名是否包含货号，确保格式正确。

**Q: 如何修改模板？**
A: 在 `templates/` 目录下放置你的 xlsx 文件，首次运行时选择即可。
